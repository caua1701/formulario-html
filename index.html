Perguntas

  1. Na tag input com atributo type="text", para que serve o atributo "placeholder"?
    1. O atributo “placeholder” na tag input com type="text" serve para fornecer um texto de ajuda dentro do campo. Este texto é exibido no campo até que o usuário comece a digitar.
  2. Na tag input com atributo type="radio" ou type="checkbox", para que serve o atributo "checked"?
    1. O atributo “checked” na tag input com type="radio" ou type="checkbox" indica que a opção correspondente deve ser selecionada por padrão quando a página é carregada.
  3. Na tag input para que serve o atributo "value"?
    1. O atributo “value” na tag input define o valor que será enviado quando o formulário for submetido.
  4. Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?
    1. Para agrupar vários botões de rádio, você deve usar o atributo “name” com o mesmo valor para todos os botões que pertencem ao mesmo grupo.
  5. Para que serve a tag label? Para que serve o atributo "for" nessa tag?
    1. A tag label serve para associar um rótulo textual a um elemento de entrada do formulário. O atributo “for” na tag label especifica o id do elemento de entrada que o rótulo representa.
  6. Na tag option para que serve o atributo "selected"?
    1. Na tag option, o atributo “selected” indica que a opção deve ser a selecionada por padrão quando a página é carregada.
  7. Nas tags de formulário para que serve o atributo "required"?
    1. Nas tags de formulário, o atributo “required” indica que um campo deve ser preenchido antes que o formulário possa ser submetido.
  8. Para que serve a tag textarea?
    1. A tag textarea é usada para criar uma área de texto multilinha, permitindo que o usuário digite texto em várias linhas.
  9. Na tag form para que serve o atributo "action"?
    1. Na tag form, o atributo “action” especifica o URL para onde os dados do formulário serão enviados após a submissão.
  10. Na tag form para que serve o atributo "method"?
    1. Na tag form, o atributo “method” especifica o método HTTP que será usado para enviar os dados do formulário (geralmente “GET” ou “POST”).
  11. Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag form?
    1. Para garantir que o valor digitado em um campo de senha não apareça na barra de endereços, o atributo “method” deve ser definido como “POST”.
  12. Qual valor deve ser informado no atributo "method" na tag form em um formulário que será usado para fazer upload de arquivos?
    1. Para fazer upload de arquivos, o valor do atributo “method” deve ser “POST”, e você também deve definir o atributo enctype como multipart/form-data.
  13. O que as tags fieldset e legend produzem?
    1. As tags fieldset e legend são usadas para agrupar elementos relacionados em um formulário e para fornecer um título para o grupo com a tag legend, respectivamente.
